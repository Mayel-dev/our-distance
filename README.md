# Our Distance 💑

API REST para una app de pareja que permite gestionar metas personales y compartidas.

## Tecnologías
- NestJS
- TypeORM
- PostgreSQL
- Docker

## Requisitos
- Node.js
- Docker Desktop

## Correr el proyecto

# Levantar la base de datos
docker-compose up -d

# Instalar dependencias
npm install

# Correr en desarrollo
npm run start:dev
